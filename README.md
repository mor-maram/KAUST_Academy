# KAUST Academy Projects 🚀

Welcome to my repository containing the projects and assignments completed during my journey at **KAUST Academy**. 

## 📁 Repository Structure

This repository is organized into independent folders, each representing a specific project or curriculum module:

*   **🤖 Machine Learning & AI:**
    *   `Neural Network with Two Layers` - Implementation of a basic neural network.
    *   `naive_bayes` - Classification using the Naive Bayes algorithm.
*   **📊 Data Science & Analytics:**
    *   `AB Testing - Average Session Duration` - Statistical analysis and Power BI visualization.
    *   `Optimization Using Gradient Descent Linear Regression` - Core optimization techniques.
*   **🧮 Mathematics & Core Foundations:**
    *   `Gaussian Elimination` - Linear algebra solver.
    *   `Application of Eigenvectors and Eigenvalues` - PCA and Webpage navigation modeling.
    *   `Linear Transformations and Neural Networks` - Geometric views of deep learning.

## 🛠️ Technologies Used
*   **Languages:** Python (`.py`, `.ipynb`)
*   **Tools:** Power BI (`.pbix`), Jupyter Notebook
*   **Libraries:** NumPy, Pandas, Scikit-Learn (and others depending on the project)
